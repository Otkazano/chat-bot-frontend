# Установка

Клонировать репозиторий:

```bash
git clone git@github.com:TechForceTurbo/frontend.git
```

Установаить зависимости:

```bash
npm install
```

Собрать проект:

```bash
npm run build
```

Запустить проект:

```bash
npm run start
```
